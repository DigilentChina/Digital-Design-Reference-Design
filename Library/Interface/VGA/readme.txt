Description:
.generate VGA signals
.resolution:640*480

Interface:
.vga_pclk:  vga work clock input(25MHz)
.vga_rst:   reset signal, when rst is '1', reset
.vga_hsync: VGA's line synchronous signal
.vga_vsync: VGA's field synchronous signal
.vga_valid: VGA's data valid signal
.vga_h_cnt: pix's column count value
.vga_v_cnt: pix's row count value
